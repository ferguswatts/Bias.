Privacy Policy

Last updated: 6 April 2026

Bias is a browser extension that displays publicly available information about New Zealand political journalists. This policy explains what data the extension does and does not collect.

Data we collect

None. The extension collects no personal data, no browsing history, no analytics, and no telemetry. We do not track which articles you read or which journalist cards you view.

Data we store

The extension stores a local copy of journalist data (names, outlets, coverage statistics, and public connections) using your browser's built-in storage. This data is sourced entirely from publicly available news articles and public records. No user data is stored.

Network requests

The extension makes a single network request on startup to fetch the latest journalist data from GitHub (raw.githubusercontent.com). If this request fails, the extension falls back to its bundled data. No other network requests are made. No data is sent from your browser to any server.

Permissions

  • Storage — used to cache journalist data locally so the extension loads quickly.
  • Host permissions for GitHub — used solely to fetch the latest journalist dataset.
  • Content script access on NZ news sites — used to detect journalist bylines and display information cards. The extension only runs on: nzherald.co.nz, stuff.co.nz, rnz.co.nz, 1news.co.nz, tvnz.co.nz, newsroom.co.nz, thespinoff.co.nz, and interest.co.nz.

Third parties

We share no data with third parties because we collect no data.

Open source

This extension is fully open source under the MIT licence. You can inspect every line of code at github.com/ferguswatts/Bias.

Contact

Questions about this policy can be directed to the project's GitHub Issues.
